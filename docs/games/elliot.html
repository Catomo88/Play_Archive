⏳ 데모판 한정 공략

데모 플레이 ★★★☆☆

이 페이지는 2026년 6월 18일 정식 발매 전 공개된 "프롤로그 데모" 기준으로 작성되었습니다. 본편 콘텐츠(4개 시대 본격 전개, 후반 던전, 보스 등)는 발매 후 추가될 예정. 데모 진행 가이드 + 시스템 이해 위주로 정리.

게임 개요

스퀘어 에닉스가 HD-2D 그래픽(옥토패스 트래블러로 시작한 도트 + 3D CG 융합 비주얼) 시리즈 최초로 만든 신작 액션 RPG. 2D 톱뷰 액션 어드벤처로, 한국 유저 평가는 "옥토패스 그래픽으로 만든 2D 젤다"가 가장 자주 나오는 표현. 영문권 리뷰에서도 "Zelda meets Octopath Traveler", "A Link to the Past + Breath of the Wild + Secret of Mana"라는 비교가 다수.

무대: 필레빌디아(Philabieldia) — 천 년에 걸친 4개의 시대(발아의 시대 등)가 펼쳐지는 세계. 주인공 엘리엇과 요정 파트너 페이(Faie) 가 함께 모험.

💡 시작 전 꼭 알아두면 좋은 점

  • 경험치/레벨 없음 — 액션 RPG라기보다 젤다식 액션 어드벤처. 강해지는 방법은 무기 / 마석 / 아이템 조합
  • 무기 7종, 동시 2개 장비 — 검·부메랑·폭탄·망치·창·활·사슬낫. 자주 바꿔쓰는 게 기본
  • 마석 시스템으로 무기 커스터마이즈 — 빌드 자유도의 핵심
  • 데모 진행은 약 1시간 30분 — 본편 데이터 연동 가능
  • 데모 평점은 80점 내외 — "타격감 약함" vs "아기자기함·재미" 양분
  • 한국어 자막 + 정식 한글화 지원

핵심 시스템 한눈에 보기

🛡️ 라이프(체력) 시스템

화면 좌상단의 빨간 물방울 마크가 체력. 게이지가 아니라 개별 라이프 개념(젤다의 하트와 동일). 회복 방법:

  • 회복의 약 — 마을 상점에서 구매
  • 모험의 길잡이 — 세계 각지에 배치. 접근하면 즉시 완전 회복 (체크포인트 + 회복소 결합)

💡 길잡이 발견 시 무조건 접근 — 다음 위험 구간 진입 전 풀체력 보장. 또한 사망 시 가까운 길잡이에서 부활.

⚔️ 무기 시스템 — 동시 2개 장비

7종 무기를 자유롭게 교체. 주무기 + 보조무기 슬롯에 1개씩 장비하고 상황에 맞게 빠르게 스왑.

무기 특성 데모 평가
검 빠른 휘두름 + 좌우 판정 ⭐ 만능, 다루기 가장 쉬움
부메랑 던진 뒤 돌아올 때까지 계속 공격 판정 유지 ⭐ 견제·다수 적 처리
폭탄 광역, 벽/장애물 파괴 보스전 기믹 필수
망치 강타·중량, 느림 강적 처치
창 사거리 ↑, 찌르기 안전 거리 유지
활 원거리 정밀 사격 ⚠️ 조준 어색 (다수 리뷰 공통 단점)
사슬낫 광역 + 끌어오기 군집 적 정리

🔮 마석 시스템

무기를 강화하는 핵심 자원.

  1. 마석 파편 획득 — 적 처치 무작위 드롭 / 보물 상자 / 도전 룸 보상
  2. 마을 마석점 방문 — 모은 파편으로 마석 생성
  3. 무기에 마석 장착 — 다양한 효과 부여 (공격력↑, 속성 추가, 특수 효과 등)
  4. 자신의 플레이 스타일에 맞는 조합 찾기 — 빌드 자유도의 핵심

🧚 페이(Faie, 요정 파트너)

엘리엇의 동반자. 솔로 모드에서는 AI가 조작, 로컬 협동에서는 다른 플레이어가 직접 조작 가능 (1인 → 2인 협동 전환).

🧚 페이의 능력 (데모 기준)

  • 마법 사격 — 자동 견제, 약한 적 직접 처치
  • 워프(Warp) — 관문 유적 보스에서 학습. 짧은 거리 순간이동 (회피·접근에 활용)
  • 부활 — 엘리엇 체력 0이 되면 페이의 마법으로 부활 (보스전에서도 유효, 매우 관대한 사망 페널티)
  • 시너지 공격 — 엘리엇 공격 + 페이 마법 조합으로 추가 데미지·이펙트

⚠️ 보스전 사망 시스템 — 보스에게 죽어도 그 자리에서 부활, 보스에게 입혔던 데미지는 그대로 유지. 처음부터 다시 시작이 아님. 공격적으로 플레이해도 손해 적음.

데모판 진행 흐름

시작 마을 → 동부 광야 탐색 → 관문 유적 진입 → 기믹·열쇠 2개 → 태그 가디언 보스 → 페이 워프 학습

데모 분량은 약 1.5시간이지만, 챌린지 룸 + 사이드 탐색 모두 클리어 시 2~3시간까지 늘어남. 본편 출시 시 데모 세이브 그대로 인계.

Chapter 1 ─ 모험가 길드와 출발

🏘️ 시작 마을 / 모험가 길드

스토리 자동 탐색 추천

필레빌디아의 거점 마을. 모험가 길드 등록, 기본 장비 수령, 페이와의 동행 시작이 이루어지는 튜토리얼 구간.

진행 단계

  1. 오프닝 컷씬 + 캐릭터 생성/선택 → 엘리엇 조작 시작
  2. 모험가 길드 방문 — NPC 대화로 첫 의뢰 수령. 페이가 합류
  3. 검 장비 + 기본 조작 튜토리얼 — 휘두르기, 회피, 점프 학습
  4. 마을 둘러보기 (선택) — 상점·여관·마석점·게시판 위치 파악
  5. 동부 광야로 출발

마을에서 챙길 것

  • 🪙 회복의 약 × 2~3개 (상점, 초기 자금 허용 범위)
  • 🗡️ 검 (기본) — 길드에서 지급
  • 📜 퀘스트 보드 확인 — 사이드 의뢰 미리 체크
  • 💎 마석점 위치 기억 — 데모 후반 마석 생성 시 재방문

💡 초보 팁 — 데모 분량은 짧지만 마을에서 NPC 대사를 모두 읽으면 본편 발매 후 시대(발아의 시대 등)에 대한 사전 정보를 얻을 수 있음. 한국 유저 후기에서 "본편 떡밥이 마을에 깔려 있다"는 평이 다수.

Chapter 2 ─ 동부 광야 (오버월드 탐색)

🌾 광활한 동부 지역

자유 탐색 챌린지 룸

관문 유적 진입 전 자유롭게 탐색 가능한 오버월드. 적 처치·상자·챌린지 룸(젤다 BotW의 사당과 유사)이 흩어져 있음. 데모 안에서도 30분~1시간 분량의 사이드 콘텐츠.

동부 광야에서 만나는 것

  • 🗡️ 비스트맨(Beastmen) — 기본 잡몹, 검·부메랑으로 처리
  • 💎 마석 파편 — 적 드롭·상자에서 무작위
  • 📦 보물 상자 — 추가 회복약·무기 시도용·마석 파편
  • 🛕 챌린지 룸(Trial) — 짧고 빠른 도전. 특정 무기 / 기믹 활용 시험. 클리어 시 보상(마석·소량 자원)
  • 🧭 모험의 길잡이 — 풀체력 회복 + 빠른 이동 거점 (추후 발견한 곳으로 워프 가능)

💡 챌린지 룸 활용 — 본편 발매 후 BotW식 "찾는 만큼 캐릭터가 강해지는" 시스템이 될 가능성 높음. 데모 단계에서 위치·풀이법을 익혀두면 본편 적응 ↑. 데모 안에서 발견 가능한 챌린지 룸은 영상 가이드(YouTube Full Demo Walkthrough) 참조.

동부 광야 ─ 추천 진행 순서

  1. 검 + 부메랑 2개 슬롯 장착 — 가장 안정적인 조합
  2. 광야 외곽 한 바퀴 → 상자·마석 파편 수집
  3. 챌린지 룸 발견 시 즉시 클리어 (데모 보상은 적지만 시스템 학습 효과 ↑)
  4. 모험의 길잡이 활성화 — 데모 안에서 발견 가능한 곳 전부
  5. 마석점 재방문 — 모은 파편으로 마석 생성, 검·부메랑에 장착
  6. 관문 유적 입구로 이동

Chapter 3 ─ 관문 유적 (Doorway Ruins)

🏛️ 데모판 메인 던전

보스 있음 전투 중심

필레빌디아 동쪽에 위치한 고대 유적. 퍼즐보다 전투 중심으로 설계되어, 전투에서 익힌 기믹을 보스전에서 활용하는 전형적 젤다식 던전 구조. 데모 안에서만 2개 던전이 진입 가능하다는 후기가 있으며, 핵심은 관문 유적.

해야 할 일: 던전 내 흩어진 열쇠 2개를 기믹으로 획득 → 봉인된 보스방 개방 → 태그 가디언 처치 → 보상.

던전 내 적 분석

⚔️

Greatsword Guardian (대검 가디언)

  • 공격: 광범위한 좌우 횡베기, 가드 단단함
  • 약점: 공격 후 백스윙 경직, 측후방 공격
  • 공략: 정면 충돌 ❌ → 부메랑으로 견제 → 빈틈에 검 콤보
  • 빠른 처리: 폭탄으로 가드 깨기 + 시너지 공격
🏹

Great Bow Guardian (대궁 가디언)

  • 공격: 장거리 정밀 사격, 차징 후 강한 일격
  • 약점: 차징 동안 이동 불가 → 측면 접근 가능
  • 공략: 엄폐물 지그재그 → 사격선 회피 → 접근 후 검 / 망치
  • 주의: 1대1 정면 사격전 = 즉사 위험

진행 단계 (관문 유적)

  1. 유적 입구 진입 — 첫 룸에서 비스트맨 처치 (튜토리얼 식)
  2. 갈림길 도달 — 좌측·우측 중 한쪽으로 진행 (어느 쪽이든 OK, 순서만 다름)
  3. 첫 열쇠 구역 — 기믹 + 가디언 1체 처치 → 열쇠 #1 획득
  4. 새 무기 발견 (선택) — 데모 도중 강화된 검 획득 가능, 보스전에 큰 도움
  5. 두 번째 열쇠 구역 — 또 다른 기믹 + 가디언 1체 → 열쇠 #2
  6. 봉인된 문 개방 → 보스방 진입 직전 모험의 길잡이 활성화 권장
  7. 태그 가디언 전투 → 클리어 시 페이의 워프 능력 학습

🎯 던전 내 수집 권장

  • 🗝️ 열쇠 ×2 (필수)
  • 🗡️ 강화된 검 (데모 보상 무기, 보스전 큰 도움)
  • 💎 마석 파편 (적 드롭)
  • 📦 보물 상자 (회복약·자원)
  • 🌀 페이의 워프 능력 (보스 처치 자동)

Chapter 4 ─ 보스: 태그 가디언 (Tag Guardians)

👹

Tag Guardians ─ 데모 최종 보스

개요: 2체가 번갈아(또는 동시에) 공격하는 태그매치 형식의 가디언 듀오. 던전에서 익힌 기믹(폭탄·페이 마법)을 종합적으로 활용해야 함.

핵심 메커니즘 (전형적 젤다식): 1. 약점/기믹 노출 페이즈 — 보스가 일정 패턴 후 약점 노출 2. 폭탄 투척 → 약점 명중 → 딜타임 발생 3. 딜타임 중 검·부메랑으로 풀딜 4. 반복

페이 활용: - 워프(보스 처치 직전 학습이라 본 보스전엔 미사용) — 데모 안에서는 보스 처치 직후 학습 - 시너지 공격 — 약점 노출 시 페이에게 마법 명령 → 추가 데미지 - 부활: 죽어도 그 자리 부활, 누적 데미지 유지 → 공격적 플레이 권장

단계별 공략

  1. 거리 유지 — 두 가디언 동시 접근 허용 X. 부메랑으로 한 쪽 견제
  2. 약점 신호 포착 — 보스 공격 패턴 끝나면 빛나는 부위 노출 (몇 초 짧음)
  3. 폭탄 투척 → 약점 명중 → 보스 경직 (딜타임)
  4. 검·강화된 검으로 풀딜 — 4~6타 가능
  5. 페이에게 마법 명령 — 약점 한 번 더 노출 / 추가 시너지 데미지
  6. 사망 시 부활 활용 — 데미지 누적 유지, 다시 공격적으로 진입
  7. 체력 1/3 이하 페이즈 — 두 가디언이 동시 협공 가능. 거리 유지 + 견제 강화

💡 보스전 효율 팁 (한국 유저 후기 종합) - 무기 슬롯: 검 + 부메랑 추천 (활은 조준 어색해서 비추) - 마석은 공격력 ↑ 우선 (시간 단축) - 회복약은 체력 1/3 이하에서만 사용 — 페이 부활을 적극 활용하면 회복약 거의 불필요 - 폭탄 잔량 확인 — 던전 진입 전 마을에서 충분히 보충

보상

  • 🌀 페이의 워프 능력 — 짧은 거리 순간이동. 본편 진행에 필수
  • 📜 데모 엔딩 시퀀스 — 본편 발매 후 이어질 스토리 떡밥
  • 💾 세이브 데이터 — 본편 데이터 연동

무기 시스템 완전 정리

엘리엇은 7종 무기를 자유롭게 운용. 주무기 + 보조 무기 2슬롯 동시 장비, R/L 키로 즉시 스왑(추정).

무기별 상세

⚔️ 검 (Sword)

기본 무기
  • 휘두름 속도: 매우 빠름
  • 공격 판정: 정면 + 좌우 (호 형태)
  • 사거리: 짧음
  • 추천 상황: 잡몹 정리, 보스 약점 풀딜, 모든 상황의 안정픽

🪃 부메랑 (Boomerang)

견제·다수 적
  • 특수성: 던지고 돌아오는 궤도 전체에 공격 판정
  • 사거리: 중거리
  • 추천 상황: 다수 적 견제, 보스 약점 페이즈 외 시간에 데미지 누적, 안전 거리 유지

💣 폭탄 (Bomb)

보스 필수
  • 특수성: 광역 데미지 + 장애물/약점 패널 파괴
  • 사용 제한: 잔량 시스템 (마을에서 보충)
  • 추천 상황: 보스 기믹 약점 명중, 군집 적 일소, 던전 막힌 길 개방

🔨 망치·창·사슬낫

  • 망치 (Hammer): 강타 + 중량. 느리지만 단발 데미지 ↑. 강적·기믹용
  • 창 (Spear): 사거리 ↑, 찌르기. 안전 거리 유지에 최적
  • 사슬낫 (Chain Scythe): 광역 + 끌어오기. 군집 정리 + 위치 조정

🏹 활(Bow) 주의 — 데모 단계에서 조준이 어색하다는 평이 다수. "맞을 것 같은데 빗나가는 경우가 잦음." 정식 발매 시 조정 가능성 있지만, 데모 안에서는 활 의존도 낮추는 것이 안정.

마석 시스템 완전 정리

  1. 마석 파편 수집 — 적 처치 무작위 드롭, 상자, 챌린지 룸 보상
  2. 마을 → 마석점 방문 — 파편을 소모해 마석 생성
  3. 인벤토리에서 무기에 장착 — 각 무기마다 슬롯 수가 있을 것으로 추정
  4. 효과 확인 후 전투 — 빌드 조정

추정되는 마석 효과 (데모 + 트레일러 기준)

  • 공격력 ↑ — 단순 데미지 증가
  • 속성 부여 — 화염·얼음·번개 등 (보스 약점 속성 매칭 시 큰 데미지)
  • 크리티컬 확률 ↑
  • 사거리 ↑ — 부메랑·창에 효과적
  • 회복 효과 — 일정 확률로 적 처치 시 체력 회복
  • 마나/스태미나 회복
  • 특수 효과 — 폭발 추가·관통 등

💡 마석 배분 우선순위 (데모 기준) 1. 공격력 ↑ — 가장 직관적이고 강함 2. 속성 — 보스 약점 알 수 있다면 매칭 3. 회복/크리 — 보조용

페이 활용 가이드

페이는 단순 NPC가 아니라 전략 파트너.

  • AI 모드 (솔로) — 자동 견제·회복. 명령으로 특정 적 공격 / 마법 발동 가능
  • 협동 모드 (2인 로컬) — 2P가 직접 조작. 더 정교한 시너지 공격
  • 시너지 공격 — 엘리엇이 약점 노출시켜 페이가 마무리 / 페이가 견제하는 사이 엘리엇이 풀딜
페이 능력 획득 시점 용도
마법 사격 시작부터 자동 견제, 약한 적 처치
워프(Warp) 관문 유적 보스 처치 짧은 거리 순간이동, 회피·접근
부활 시작부터 엘리엇 체력 0 시 자동 부활 (강력)
시너지 시작부터 약점 노출·강력한 콤보 발동

데모 평가 (한국 유저 후기 종합)

👍 호평 포인트

  • HD-2D 그래픽의 아기자기함 — 옥토패스 트래블러 팬에게 매우 익숙하고 좋은 인상
  • 2D 젤다 향수 — 과거 2D 액션 RPG·젤다 팬에게 강한 어필
  • 퍼즐+전투 균형 — 던전이 단순 미로가 아니라 전투 학습 → 보스 활용 구조
  • 페이 부활 시스템 — 보스전 좌절 ↓, 공격적 플레이 유도
  • 무기 7종 시스템 — 정식 발매 시 빌드 자유도 기대
  • 체험판 분량 — 1.5시간 + 사이드 합치면 2~3시간, 부담 없이 본편 맛보기 가능

👎 아쉬운 점 (데모 기준) - 활 조준 어색 — 다수 후기 공통 단점. 정식 발매 시 패치 기대 - 액션 타격감 약함 — "아기자기하지만 묵직함 부족" - 액션 스피드 약간 느림 — 정통 ARPG보다 어드벤처 쪽에 가까움 - 경험치·레벨 없음 — RPG 성장 기대했다면 실망 가능

전투 핵심 팁

  1. 검 + 부메랑 = 만능 조합 — 솔로 플레이 안정픽
  2. 회피 우선 — 가드는 약하고 회피 윈도우가 넉넉
  3. 점프 활용 — 일부 적 공격은 수직 회피 가능
  4. 페이 부활 적극 활용 — 한 번 죽어도 누적 데미지 유지 → 보스에 공격적으로
  5. 모험의 길잡이 = 회복 + 빠른 이동 거점 — 위험 구간 전 무조건 활성화
  6. 챌린지 룸은 보너스 컨텐츠 — 본편 강화 자원 확보 + 시스템 학습
  7. 폭탄 잔량 관리 — 보스 직전엔 풀로

자주 묻는 질문

Q. Switch 2 한정 데모 아닌가? 다른 플랫폼에서도 가능? A. 초기에는 Switch 2 한정으로 공개됐지만, 이후 모든 발매 플랫폼(PS5/XSX/PC)에서 한국어판 데모 배포가 진행 중. 자세한 일정은 스퀘어 에닉스 공식 또는 PS Store 페이지 확인.

Q. 데모 세이브가 본편에 인계되나? A. Yes. "프롤로그 데모"는 명시적으로 본편 데이터 연동 지원. 단, 동일 플랫폼에서만 연동 가능.

Q. 1인용? 협동? A. 솔로 + 로컬 협동 2인. 페이를 2P가 조작. 온라인 협동은 미발표.

Q. 한국어 더빙? A. 미발표. 자막 한국어화 확정. 음성은 영어/일본어 선택 가능.

Q. 옥토패스 트래블러 후속작? A. 아닙니다. HD-2D 그래픽 엔진은 동일하지만 세계관·시리즈 무관. 옥토패스는 턴제 RPG, 엘리엇은 액션 어드벤처.

Q. 4개의 시대란? A. 발아의 시대 등 천 년에 걸친 4개 시대가 본편 전개의 핵심. 데모는 가장 첫 시대(=발아의 시대) 일부만 다룸. 시간 여행 / 시대 분기가 본편 시스템.

Q. 본편 분량 예상? A. 공식 발표 없음. 4개 시대 × 데모 분량(1.5h)을 단순 곱해도 6h+이지만, 시대당 던전·사이드 합치면 본편 30~50시간 추정.
